Q1. 「RUNTEQはプログラミングスクールである」の命題が真であるとする。

以下の命題で真となるものはどれか。
1. 「プログラミングスクールはRUNTEQである」
2. 「RUNTEQでないならプログラミングスクールではない」
3. 「プログラミングスクールでないならRUNTEQではない」

※ 元は「雪は白い」が命題で
1. 「白いものは雪である」逆 --> 偽
2. 「雪でないなら白くない」裏 --> 偽

上記が正しいのですが、現実では黒く濁った雪などもあり得るので対偶が真だと選びづらいと感じたため、プログラミングスクール関連の問題にしてみました。
